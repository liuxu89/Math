## `Gamma`和`Beta`函数

### ``\Gamma``函数
```math
\Gamma(\alpha) := \int_0^{+\infty} t^{\alpha-1} e^{-t} \mathrm{d} t \quad (\alpha > 0)
```
满足

```math
\Gamma(1) = 1; \quad \Gamma(\frac{1}{2}) = \sqrt{\pi} \\
\Gamma(\alpha+1) = \alpha \Gamma(\alpha); \quad \Gamma(n) = (n-1)!
```
